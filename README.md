# Buffet-Bro
Windows PowerShell Text-Game

I created this game to increase my proficiency in Windows PowerShell.<br/>
This really helped me learn:
* Functions
* Switches
* While Loops
* If statements
* Global Variables
* String Manipulation
* Progress Output
